Project Name=> Digitool 




Description=> This is a modern web application. This platform allows users to add the products to their card and simulates a simple checkout experience.


Used technologies :
React
JS
CSS
JSON


Key features =>
Product and card system: Users can add items to the card remove their item and checkout with real time notification at the top.

Good UI with multiple section: Includes a very beautiful responsive layout that has banner CTA sections and many more .
